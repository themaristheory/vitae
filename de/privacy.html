Datenschutzerklärung

Zuletzt aktualisiert: Juli 2026

Vitae ist eine Local-First-App. Alle Gesundheitsdaten, die Sie eingeben, werden ausschließlich auf Ihrem Gerät gespeichert. Vitae hat keine Benutzerkonten; die Entwicklerin kann Ihre Daten weder einsehen noch darauf zugreifen oder sie wiederherstellen.

Welche Daten erhebt Vitae?

Keine. Vitae erhebt, überträgt und verarbeitet keine personenbezogenen Daten. Die App enthält keine Analyse-, Werbe- oder Tracking-Dienste.

Wo werden meine Daten gespeichert?

Alle Daten werden lokal in einer Datenbank auf Ihrem Gerät gespeichert. Sie verlassen Ihr Gerät nur, wenn Sie sie selbst exportieren oder ein Backup erstellen.

Backups

Backups sind optional und liegen vollständig in Ihrer Hand. Wenn Sie das Cloud-Backup aktivieren, wird die Backup-Datei in Ihrem persönlichen iCloud-Konto (iOS) bzw. Google-Drive-Konto (Android) gespeichert. Die Entwicklerin hat keinen Zugriff auf Ihre Backups; Sie können diese jederzeit in Ihrem eigenen Cloud-Speicher löschen.

Trinkgeld

Vitae bietet freiwillige Trinkgelder als In-App-Käufe an. Die Zahlungsabwicklung erfolgt vollständig über Apple bzw. Google gemäß deren Datenschutzbestimmungen. Die Entwicklerin erhält keine personenbezogenen Daten oder Zahlungsdaten.

Aktivierung kostenpflichtiger Vitae-Funktionen

Wenn Sie Vitae+ mit einem Zugangsschlüssel aktivieren, wird ausschließlich dieser Schlüssel zur Überprüfung an unseren Server übermittelt. Dabei werden keine Gesundheitsdaten oder personenbezogenen Informationen übertragen.

Website

Diese Website wird über GitHub Pages gehostet. GitHub kann aus technischen und Sicherheitsgründen IP-Adressen der Besucher protokollieren — Näheres finden Sie in der Datenschutzerklärung von GitHub. Die Vitae-Website selbst verwendet keine Cookies und keine Analysetools.
