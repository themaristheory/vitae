Hi, welcome to Vitae's guide!

My name is Mariana, and I built Vitae. In these pages I'll walk you through everything the app can do.

Vitae is your personal health history — appointments, exam results, medications, symptoms, and more — stored entirely on your phone. No account, no internet connection needed.



## Installation

Right after installing Vitae, you have two options: start fresh, or load from a backup file you created before.

Even if you start fresh, you can still load a backup file later at any time.

### Fresh start

If you are starting fresh, you can fill in your information right away or skip the whole thing and go straight to the app's first page.
Everything asked during onboarding can also be added or edited later inside the app, and it's used only to fill the SOS page.

### Loading from a backup file

If you are reinstalling Vitae, you may already have a backup file saved on your phone. Restoring from it will **replace** everything currently in the app with the contents of the file. It is a full replacement, not a merge, so anything added after the backup was created will be lost.



## Using the app

When you reach Vitae's first page, a guided tour will show you around.
The tour appears on some pages to help you find your way and discover what is possible.

Skipped the tour for whatever reason? You can turn it back on in Settings. Check it out!
